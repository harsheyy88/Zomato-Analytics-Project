# Zomato Analytics & Business Intelligence Project

## Project Overview

This project presents an end-to-end data analytics solution using Zomato restaurant data. The analysis combines Python-based exploratory data analysis, customer and restaurant segmentation, predictive analytics, and an interactive Power BI dashboard to uncover meaningful business insights related to ratings, pricing, cuisines, delivery services, and customer preferences.

The project demonstrates the complete analytics workflow, from data preparation and statistical analysis to visualization and business intelligence reporting.

---

## Project Workflow

### 1. Data Collection & Preparation
- Imported and examined Zomato dataset
- Cleaned missing and inconsistent values
- Standardized data formats for analysis

### 2. Exploratory Data Analysis (EDA)
- Analyzed restaurant ratings and customer trends
- Examined pricing patterns across restaurants
- Investigated cuisine popularity and distribution
- Identified key performance indicators

### 3. Data Segmentation & Predictive Analysis
- Performed segmentation to identify restaurant categories and customer behavior patterns
- Applied analytical techniques to uncover trends and support business decision-making
- Generated insights for strategic recommendations

### 4. Business Intelligence Dashboard Development
- Built an interactive Power BI dashboard
- Created KPI cards and performance metrics
- Developed visualizations for ratings, pricing, cuisines, and restaurant distribution
- Enabled dynamic filtering and data exploration

---

## Key Insights

- Restaurant ratings vary significantly across price categories.
- Certain cuisines consistently achieve higher customer satisfaction.
- Pricing and customer ratings exhibit identifiable patterns.
- Segmentation analysis helps classify restaurants into meaningful business groups.
- Interactive dashboards improve decision-making through visual analytics.
